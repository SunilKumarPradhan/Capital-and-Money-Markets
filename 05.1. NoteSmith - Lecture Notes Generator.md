## 📘 What Are Options? - The Fundamentals

### Definition

> **Option**: A derivative contract that grants the holder the **right, but not the obligation**, to buy or sell an underlying asset at a predetermined price (strike price) before a specified expiration date.

### The House Analogy
Think of an option like having the right to purchase a house at a fixed price in the future:
- You pay a non-refundable deposit (premium)
- You can buy the house at the agreed price (strike price) anytime before a deadline (expiration)
- If house prices rise, you benefit by buying at the lower agreed price
- If house prices fall, you simply walk away, losing only your deposit

### Why Options Exist

**Problem They Solve:**
1. **Uncertainty Management**: Future prices are unpredictable
2. **Capital Efficiency**: Control assets without full ownership
3. **Asymmetric Risk**: Limited downside, unlimited upside potential
4. **Strategic Flexibility**: Create custom risk-reward profiles

**When to Use Options:**
- **Hedging**: Protect existing positions from adverse moves
- **Speculation**: Bet on price direction with limited risk
- **Income**: Generate cash flow from premium collection
- **Arbitrage**: Exploit pricing inefficiencies

### The Two Parties in Every Option Trade

| Party | Role | Rights | Obligations | Risk Profile |
|-------|------|--------|-------------|--------------|
| **Buyer (Holder)** | Pays premium | Can exercise option | None (can let expire) | Limited to premium paid |
| **Seller (Writer)** | Receives premium | None | Must fulfill if exercised | Potentially unlimited |

## 🔑 Key Features of Options

### 1. Strike Price (Exercise Price)

**Definition**: The predetermined price at which the option holder can buy or sell the underlying asset.

**Importance:**
- Determines potential profitability
- Affects premium pricing
- Defines risk-reward profile

**Example:**
```
Stock Current Price: $55
Call Option Strike: $50

If exercised:
- Buy stock at $50 (strike price)
- Sell immediately at $55 (market price)
- Profit: $5 per share (minus premium paid)
```

### 2. Option Premium

**Definition**: The price paid by the buyer to the seller for the option contract.

**Components:**

```
Total Premium = Intrinsic Value + Extrinsic Value
```

#### Intrinsic Value
The immediate profit if exercised right now:

**For Call Options:**
```python
intrinsic_value = max(0, current_price - strike_price)

# Example:
current_price = 60
strike_price = 50
intrinsic_value = max(0, 60 - 50) = 10
```

**For Put Options:**
```python
intrinsic_value = max(0, strike_price - current_price)

# Example:
current_price = 40
strike_price = 50
intrinsic_value = max(0, 50 - 40) = 10
```

#### Extrinsic Value (Time Value)
The additional premium beyond intrinsic value, accounting for:
- **Time to expiration**: More time = more value
- **Volatility**: Higher volatility = higher premium
- **Interest rates**: Affects present value calculations
- **Dividends**: Expected payouts reduce call value, increase put value

**Factors Influencing Premium:**

| Factor | Impact on Call Premium | Impact on Put Premium |
|--------|----------------------|---------------------|
| ↑ Underlying Price | Increases | Decreases |
| ↑ Strike Price | Decreases | Increases |
| ↑ Time to Expiration | Increases | Increases |
| ↑ Volatility | Increases | Increases |
| ↑ Interest Rates | Increases | Decreases |
| ↑ Dividends | Decreases | Increases |

### 3. Expiration Date

**Critical Characteristics:**
- Options have **limited life**
- Value erodes as expiration approaches (time decay)
- Worthless if not exercised by expiration (for out-of-the-money options)
- Creates urgency and risk

**Time Decay Acceleration:**
```
Time decay is NOT linear - it accelerates as expiration approaches

Example timeline for a 90-day option:
Days 90-60: Slow decay (~20% of time value)
Days 60-30: Moderate decay (~30% of time value)
Days 30-0:  Rapid decay (~50% of time value)
```

## 📊 Types of Options

### Call vs. Put Options

| Feature | Call Option | Put Option |
|---------|------------|------------|
| **Right Granted** | Buy the asset | Sell the asset |
| **Buyer Outlook** | Bullish (expects price rise) | Bearish (expects price fall) |
| **Profit When** | Price > Strike | Price < Strike |
| **Maximum Loss** | Premium paid | Premium paid |
| **Maximum Gain** | Unlimited | Strike - Premium (limited to strike) |

### Call Option Example

```python
# Call Option Details
strike_price = 50
premium_paid = 3
current_stock_price = 60

# At Expiration Scenarios:

# Scenario 1: Stock at $60 (In the Money)
exercise_profit = current_stock_price - strike_price  # 60 - 50 = 10
net_profit = exercise_profit - premium_paid  # 10 - 3 = 7 per share

# Scenario 2: Stock at $45 (Out of the Money)
exercise_profit = 0  # Don't exercise
net_profit = -premium_paid  # -3 per share (loss limited to premium)

# Breakeven Point
breakeven = strike_price + premium_paid  # 50 + 3 = 53
# Stock must reach $53 to break even
```

### Put Option Example

```python
# Put Option Details
strike_price = 50
premium_paid = 4
current_stock_price = 35

# At Expiration Scenarios:

# Scenario 1: Stock at $35 (In the Money)
exercise_profit = strike_price - current_stock_price  # 50 - 35 = 15
net_profit = exercise_profit - premium_paid  # 15 - 4 = 11 per share

# Scenario 2: Stock at $55 (Out of the Money)
exercise_profit = 0  # Don't exercise
net_profit = -premium_paid  # -4 per share

# Breakeven Point
breakeven = strike_price - premium_paid  # 50 - 4 = 46
# Stock must fall to $46 to break even
```

### European vs. American Style Options

| Feature | European Style | American Style |
|---------|---------------|----------------|
| **Exercise Timing** | Only at expiration | Anytime before expiration |
| **Flexibility** | Lower | Higher |
| **Premium** | Lower | Higher (due to flexibility) |
| **Common Markets** | Index options, some equity | Most equity options, ETFs |
| **Pricing Complexity** | Simpler | More complex |

**Why the Difference Matters:**

**European Options:**
- Easier to price (Black-Scholes works perfectly)
- Lower time value component
- Preferred for index options
- Settlement typically cash-based

**American Options:**
- Early exercise possible (important for dividends)
- Higher premium justified by flexibility
- More common in US equity markets
- Can be physically settled

**When Early Exercise Makes Sense:**
1. **Deep in-the-money calls** before ex-dividend date
2. **Deep in-the-money puts** when carrying cost exceeds time value
3. **Avoiding assignment risk** on short positions

### Exotic Options (Brief Overview)

**Asian Options:**
- Payoff based on average price over period
- Reduces manipulation risk
- Lower volatility than standard options

**Bermudian Options:**
- Exercise allowed on specific dates (not anytime)
- Hybrid between European and American
- Common in interest rate markets

**Barrier Options:**
- Activate or deactivate at certain price levels
- **Knock-In**: Becomes active when barrier reached
- **Knock-Out**: Becomes worthless when barrier reached
- Lower premiums than standard options

## 💰 Understanding Option Premiums

### Premium Pricing Formula (Conceptual)

```
Premium = Intrinsic Value + Time Value + Volatility Premium
```

### Detailed Premium Components

#### 1. Intrinsic Value
**What it represents:** Immediate exercise value

```python
def calculate_intrinsic_value(option_type, spot_price, strike_price):
    """
    Calculate intrinsic value for call or put option
    
    Parameters:
    option_type: 'call' or 'put'
    spot_price: current market price of underlying
    strike_price: option strike price
    
    Returns:
    intrinsic_value: non-negative value
    """
    if option_type == 'call':
        intrinsic_value = max(0, spot_price - strike_price)
    elif option_type == 'put':
        intrinsic_value = max(0, strike_price - spot_price)
    else:
        raise ValueError("option_type must be 'call' or 'put'")
    
    return intrinsic_value

# Examples:
print(calculate_intrinsic_value('call', 55, 50))  # Output: 5
print(calculate_intrinsic_value('call', 45, 50))  # Output: 0
print(calculate_intrinsic_value('put', 45, 50))   # Output: 5
print(calculate_intrinsic_value('put', 55, 50))   # Output: 0
```

#### 2. Time Value
**What it represents:** Value from possibility of favorable price movement

**Time Value Calculation:**
```python
def calculate_time_value(option_premium, intrinsic_value):
    """
    Time value is the extrinsic component of premium
    
    Parameters:
    option_premium: total market price of option
    intrinsic_value: calculated intrinsic value
    
    Returns:
    time_value: premium beyond intrinsic value
    """
    time_value = option_premium - intrinsic_value
    return max(0, time_value)  # Cannot be negative

# Example:
option_premium = 7.50
intrinsic_value = 5.00
time_value = calculate_time_value(option_premium, intrinsic_value)
print(f"Time Value: ${time_value}")  # Output: $2.50
```

**Time Decay (Theta):**
```python
import numpy as np
import matplotlib.pyplot as plt

def illustrate_time_decay():
    """
    Demonstrate how time value erodes as expiration approaches
    """
    days_to_expiration = np.arange(90, -1, -1)
    
    # Time value decay (accelerates near expiration)
    time_value = 5 * np.sqrt(days_to_expiration / 90)
    
    return days_to_expiration, time_value

# Visualization concept:
# Day 90: Time Value = $5.00
# Day 45: Time Value = $3.54 (29% decay)
# Day 10: Time Value = $1.67 (67% decay)
# Day 1:  Time Value = $0.53 (89% decay)
# Day 0:  Time Value = $0.00 (100% decay)
```

#### 3. Volatility Premium
**What it represents:** Compensation for uncertainty

**Higher Volatility = Higher Premium because:**
- Greater probability of large price swings
- More chance option ends in-the-money
- Increased risk for option seller

```python
def volatility_impact_on_premium(base_premium, volatility_change):
    """
    Simplified illustration of volatility impact
    
    Parameters:
    base_premium: current option premium
    volatility_change: percentage change in implied volatility
    
    Returns:
    new_premium: adjusted premium
    """
    # Vega approximation: $0.10 per 1% volatility change (simplified)
    vega = 0.10
    premium_change = vega * volatility_change
    new_premium = base_premium + premium_change
    
    return new_premium

# Example:
current_premium = 5.00
vol_increase = 5  # 5% increase in implied volatility

new_premium = volatility_impact_on_premium(current_premium, vol_increase)
print(f"Premium after volatility increase: ${new_premium}")
# Output: $5.50 (increased by $0.50)
